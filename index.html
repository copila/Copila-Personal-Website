Projects

Keymochi

Keymochi is an iOS application that can detect emotion based on keystroke pattern. The keyboard can be used universally across all applications to capture a hollistic view of a person's mood. We extract the key features out of the data (speed of typing, distance between keys, punctuation use, rudimentary sentiment analysis, gyration, and acceleration), and discard the actual content. We then use machine learning to detect emotion.

Slamble

This application sought to improve sleeping habits by creating a game with friends. The web component used HTML, CSS, JavaScript, Backbone.js, EJS, and Parse for the backend. The iOS application used Objective C for the frontend, JavaScript for middleware and APIs, and Parse for the backend.

Flow

Flow is an application that enables knowledge workers to maintain focus and improve productivity. With our Mac and iPhone application “Flow,” users in a state of concentration receive their notifications (text, email, Slack) in the form of occasional digests. The iPhone application is visibly propped up, showing colleagues that the user does not want to be disturbed. For users not “in flow,” they are able to see statuses of other team members so they know who is currently unavailable to chat. This workflow carves out the space and time for knowledge workers to truly be productive, while still staying engaged in the workplace.
